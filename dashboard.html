Crime Rate Choropleth Map

This choropleth map returns the r-values and p-values for the correlation between unemployment and crime rate. The drop down menu will allow you to filter for each type of crime, and you can click on each state to get the r-value and p-value for that state. Additionally, states that are red indicate a negative correlation, and states that are green indicate a positive correlation.

Bubble Map

This bubble map gives visual representation to crime rates and unemployment rates, compared to population. Larger state populations will be represented by larger circles, whereas smaller circles denote a smaller state population. Ligter colored circles indicate low crime rates and unemployment rates, with circles getting darker as those values increase. The user can click through different years and crimes to see the map change. Users can also select each bubble for a breakdown of state name, population size, crime rate, and unemployment rate.
